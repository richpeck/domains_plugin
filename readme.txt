Custom plugin to manage domain names inside andy.co.uk

== Description ==

Custom plugin to help manage the domains inside andy.co.uk

Plugin creates a "domain" CPT and accompanying taxonomy, both of which are used to provide the means to manage domains in the backend.

The system was designed to bring in the domains into a single website and then have them displayed for purchase.
